#### 方法一：position绝对定位，适用于最外层
```css
.box {
	width: 100px;
	height: 100px;
	background-color: red;
	position: absolute;
	left: 50%;
	top: 50%;
	transform: translate(-50%, -50%);
}
```

#### 方法二：display:flex 适用于外层有div包裹的内层居中
```css
.content {
	width: 200px;
	height: 200px;
	background-color: red;
	display: flex;
	justify-content: center;
	align-items: center;
}

.content .item {
	width: 50px;
	height: 50px;
	background-color: black;
}
```

#### 方法三：auto margins
```css
.parent { display: flex } 
.child { margin: auto; }
```
